# LinkedIn Jobs

## Outcome
Build a repeatable, evidence-based LinkedIn job search workflow that improves role fit and application quality.

## Workflow
1. Capture target constraints before searching.
2. Generate focused LinkedIn searches and filters.
3. Score each role and rank the shortlist.
4. Tailor application artifacts to top roles.
5. Draft outreach and follow-up messages.
6. Maintain execution cadence with a tracker.

## 1) Capture Target Constraints
Collect and lock these inputs first:
- Target titles: primary + adjacent titles.
- Seniority: level and minimum years.
- Location: remote, hybrid, on-site, commute radius.
- Compensation: floor, preferred range, bonus/equity expectations.
- Tech/domain priorities: must-have and nice-to-have.
- Deal breakers: visa, travel %, industry exclusions, shift timing.

If inputs are missing, infer a practical default and proceed.

## 2) Generate LinkedIn Searches
Use [linkedin-search-playbook.md](references/linkedin-search-playbook.md).

Rules:
- Start with 2 to 4 tightly scoped queries.
- Use one intent per query (role, stack, or domain), then combine only after reviewing results.
- Prefer fresh jobs first (`Past week` or `Past 24 hours`) for higher response probability.
- Maintain a "high-fit only" pass and a separate "stretch" pass.

## 3) Score and Rank Roles
Use [fit-scoring.md](references/fit-scoring.md).

Rules:
- Score every shortlisted role before drafting applications.
- Reject roles under 60 unless the user explicitly wants stretch opportunities.
- Prioritize top 5 to 10 roles by score and freshness.

When a job description is incomplete, mark assumptions explicitly.

## 4) Tailor Application Artifacts
For each top role:
- Rewrite resume summary to align with role outcomes.
- Reorder bullets to match top requirements.
- Add 3 to 6 keyword-aligned terms naturally (no stuffing).
- Draft a concise role-specific cover message when requested.

Use templates from `assets/`:
- `assets/job-shortlist-template.md`
- `assets/application-customization-template.md`

## 5) Draft Outreach
Create two messages per role:
- Recruiter message (quick fit + ask)
- Hiring manager message (outcome-led value pitch)

Constraints:
- Keep each message under 120 words.
- Include one concrete relevance proof.
- Use one clear CTA.

Use `assets/outreach-templates.md`.

## 6) Run Weekly Cadence
Operate in weekly cycles:
- Identify new jobs daily.
- Submit applications in batches.
- Send outreach within 24 hours of applying.
- Follow up after 5 to 7 days if no response.

Track status with `assets/job-shortlist-template.md`.

## Quality Bar
- Favor fit and quality over application volume.
- Avoid generic language that could apply to any role.
- Surface risks early: missing skill evidence, location mismatch, compensation mismatch.
- Show assumptions and confidence for each recommendation.
